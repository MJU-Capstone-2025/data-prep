### RandomForest - Target: `Coffee_Price`

| rf_price_v1.0.0 | 범주형 변수에 OneHotEncoding을 적용한 후 커피 가격 자체를 예측. 커피 가격은 장기적으로 상승 추세를 보이나, RandomForest는 평균 회귀 경향이 강해 낮은 가격대 중심으로 예측됨. 따라서 테스트 구간에서는 첫날 실제 가격으로 예측값을 보정함. |
